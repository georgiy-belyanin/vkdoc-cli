# Vkdoc CLI

This is a supplementary utility tool for converting various documentation
formats into [VK Docs](https://github.com/vk-cs/docs-public/blob/master/guides/how-it-works.md)
using [Pandoc](https://github.com/jgm/pandoc).

## Dependencies

* [Node](https://nodejs.org/en)
* [Pandoc](https://github.com/jgm/pandoc)

## Building & using

You can build the project using `npm`.

```bash
npm i && npm run build
```

Run Vkdoc CLI using `node`.

```bash
node ./dist/main.js
```
